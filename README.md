# DL_26_01_Team1
딥러닝을 이용한 쓰레기 자동 분류 및 분리배출 도우미

> AI Hub 공공데이터와 ResNet-50 전이 학습을 활용한 실시간 폐기물 재질 판별 웹 서비스

1. 프로젝트 개요
* 배경: 복잡한 국내 분리배출 규정으로 인한 시민들의 혼란을 해소하고 자원 순환율을 높이기 위함.
* 목적: 스마트폰으로 쓰레기를 촬영하면 실시간으로 5대 재질을 분류하고 올바른 배출 가이드를 제공하는 엔드-투-엔드 웹 서비스 구현.

2. 주요 기술 스택
* Language: Python 3.12
* Deep Learning: PyTorch, Torchvision
* Architecture: ResNet-50 (Pre-trained)
* Web UI: Streamlit, ngrok
* Environment: Google Colab (T4 GPU)

3. 핵심 엔지니어링 포인트
* 데이터 증강: 학습 과정 중 무작위 좌우 반전 및 15도 회전을 주입하여 모델의 과적합을 방지하고 일반화 성능 확보.
* 검증 연동형 최적 가중치 저장: 에포크마다 Train/Val 성능을 모니터링하여, 과적합 발생 직전 가장 성능이 우수했던 6 에포크 시점의 최적 가중치(Best Accuracy: 91.53%)를 자동 복원.

4. 향후 고도화 계획
* 이미지 분류 모델의 배경 노이즈 취약성을 극복하기 위해, 물체의 정밀 구역만 탐색하는 YOLO 기반의 객체 탐지 파이프라인으로 전환 예정.
